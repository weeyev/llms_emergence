# LLM Emergence Literature Audit

This repository contains the source corpus, tiered paper audits, and static dashboard for the LLM emergence literature review.

Shared dashboard:

https://weeyev.github.io/llms_emergence/

Primary local entrypoint:

- `EMERGENCE_AUDIT_DASHBOARD.html`

Core report files:

- `FINAL_REPORT.md`
- `FINAL_PAPER_LEVELS.md`
- `LEVEL1_SURPRISE_AUDIT.md`
- `GUTH_MENARD_CRITERIA_AUDIT.md`
- `RAINBOW_JMLR_CRITERIA_AUDIT.md`
- `research_sources/FINAL_PAPER_LEVELS.tsv`
- `research_sources/FINAL_PAPER_LEVELS.json`
- `research_sources/LEVEL1_DOI_INDEX.tsv`
- `research_sources/LEVEL1_DOI_INDEX.json`

The audit treats Level 1 as a review-priority label, not a verified-emergence conclusion. Final acceptance depends on full-paper checks for methodological fit, source availability, claim specificity, evidence grounding, and whether the paper supports compact causal or predictive structure rather than only broad capability scaling.
